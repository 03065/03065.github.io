Как отключить автоматическое обновление с помощью реестра на Windows Server 2019
  1. Нажмите сочетание клавиш Windows + R, чтобы открыть окно команды «Выполнить»
  2. Введите regedit и нажмите OK, чтобы открыть редактор реестра.
  3. Перейдите по следующему пути: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows
  4. Кликните правой кнопкой мыши по разделу Windows и выберите Создать > Раздел.
  5. Назовите новый раздел WindowsUpdate и нажмите Enter.
  6. Кликните правой кнопкой мыши по созданному разделу WindowsUpdate и выберите Создать > Раздел.
  7. Назовите новый раздел AU и нажмите Enter.
  8. Кликните правой кнопкой мыши по созданному разделу AU и выберите Создать > Параметр DWORD (32 бита).
  9. Назовите новый параметр NoAutoUpdate и нажмите Enter.
  10. Дважды кликните по созданному параметру и поменяйте значение с 0 на 1.
  11. Нажмите кнопку OK.

В любой момент вы можете откатить изменения, используя ту же последовательность действий, но на шаге 4 нужно удалить раздел WindowsUpdate, используя меню правой кнопки мыши. Затем перезагрузите устройство, чтобы применить изменения
